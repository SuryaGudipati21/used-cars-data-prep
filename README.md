# Car Price Prediction - Data Preprocessing & Baseline Model

## Overview
This project focuses on cleaning and preparing the CarDekho used car dataset for machine learning. It includes handling missing values, feature engineering, encoding categorical variables, and evaluating a baseline model.

## Steps Performed
- Data cleaning (handling missing values, removing outliers)
- Feature engineering (converting mileage, engine, max_power to numeric)
- Encoding categorical variables (label encoding & one-hot encoding)
- Train-test split (80-20)
- Baseline model using mean prediction
- Evaluation using Mean Absolute Error (MAE)

## Baseline Model
A simple baseline was created by predicting the mean of training data for all test samples.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Evaluation Metric
- Mean Absolute Error (MAE)

## Goal
To establish a clean dataset and baseline performance before applying machine learning models.
